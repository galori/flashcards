# Flash Cards App

A self-contained, single-page flashcard study app with no backend or dependencies — everything lives in `index.html`.

**Features:**
- Create and manage decks of term/definition cards, optionally with images
- Import cards directly from a Quizlet page (paste the page source)
- Six study modes: Flashcards (flip), Multiple Choice, True or False, Type It, Starred Only, and Spaced Review
- Star cards to focus on weak spots; spaced review orders cards by lowest score first
- Progress tracking with per-card score history, stored in `localStorage`
- Themeable UI (light/dark and color themes), shuffle toggle, and auto-advance option

# Files

* test/quizlet_SAMPLE_HTML_SOURCE_FOR_IMPORT.html - test snapshot of a quizlet page, can be used for importing
